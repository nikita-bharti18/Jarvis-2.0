# Jarvis 2.0 - AI Voice Assistant

Jarvis 2.0 is a simple AI-based Voice Assistant built using Python.
It can perform tasks like opening websites, searching Wikipedia, telling time, and responding to voice commands.

---

# Features

* Voice Recognition
* Text to Speech
* Open Google
* Open YouTube
* Open GitHub
* Search Wikipedia
* Tell Current Time
* Open VS Code
* Exit with Voice Command

---

# Technologies Used

* Python
* SpeechRecognition
* pyttsx3
* Wikipedia API
* Webbrowser Module

---

# Installation

## Clone Repository

```bash id="i5fh6q"
git clone https://github.com/rishi02soni/Jarvis-2.0.git
```

## Move to Project Folder

```bash id="f2b4hz"
cd Jarvis-2.0
```

## Install Required Libraries

```bash id="hbrlv2"
pip install pyttsx3 SpeechRecognition wikipedia pyaudio
```

If pyaudio gives error:

```bash id="1xx87l"
pip install pipwin
pipwin install pyaudio
```

---

# Run the Project

```bash id="i9i0b4"
python jarvis.py
```
